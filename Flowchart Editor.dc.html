Cara menulis diagram

Diagram ditulis baris demi baris dari atas ke bawah. Ada empat jenis baris:

[Teks]Sebuah kotak (bahan / produk / hasil). Contoh: [Serbuk sampel]
- TeksSebuah perlakuan, muncul sebagai garis kecil + teks di kanan garis alir.
< TeksSebuah penambahan reagen (tanda panah masuk ke garis). Contoh: < H₂O (V = 500 mL)
splitMembuat percabangan. Di bawahnya diberi beberapa blok branch (menjorok ke dalam).

Percabangan ditulis dengan split lalu tiap cabang diawali branch dan isinya menjorok masuk (indentasi 2 spasi tiap level):

[Filtrat]
split
  branch
    [Supernatan]
    - Pengukuran absorbansi
    [Hasil]
  branch
    [Endapan]

Untuk memaksa ganti baris di dalam kotak atau perlakuan yang panjang, sisipkan \n (garis miring + n). Simbol seperti ₂ ₃ ² λ ° boleh diketik langsung.

Format teks

Bisa diketik langsung atau lewat tombol B / I / U / X₂ / X² di atas editor. Teks dasar tetap normal (tidak tebal).

**teks**Tebal
*teks*Miring
__teks__Garis bawah (dua garis bawah)
~teks~Subskrip — mis. H~2~O, Fe~2~O~3~
^teks^Superskrip — mis. cm^2^, Cu^2+^

Untuk menampilkan simbol markup secara literal, awali dengan \ (mis. \* menghasilkan tanda bintang biasa).

Preview diperbarui otomatis. Kesalahan sintaks ditampilkan di bawah editor beserta nomor barisnya.
